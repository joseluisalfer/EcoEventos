# ProyectoMinsait 💻

Repositorio para el proyecto FCT de **Minsait**, centrado en el desarrollo de una aplicación web llamada **EcoEventos** 🌱.

---

## 🌱 Objetivos del proyecto

- ✅ Creación de la página web de **EcoEventos**
- ✅ Diseño y desarrollo de la base de datos relacional
- ✅ Documentación completa del proceso del proyecto
- ✅ Desarrollo de clases en C# aplicando Programación Orientada a Objetos

---

## 🌍 🌐 Visitar la web

👉 Puedes acceder a la versión web del proyecto aquí:

🔗 **[EcoEventos Web](https://ecoeventos.netlify.app/)**

En esta web podrás ver la interfaz desarrollada con HTML y CSS, diseñada para ofrecer una experiencia sencilla, visual y accesible del proyecto EcoEventos.

---

## 📁 Estructura del proyecto

### 🔖 Lenguaje de Marcas
Creación de la página web estática de EcoEventos utilizando **HTML y CSS**, incluyendo:

- Maquetación de páginas principales
- Uso de etiquetas semánticas
- Diseño visual acorde a temática ecológica 🌱

---

### 🗃️ Base de Datos
Desarrollo de base de datos en **MySQL**, que gestiona:

- Usuarios, eventos, categorías, ubicaciones e inscripciones
- Relaciones mediante claves primarias y foráneas
- Scripts de creación y datos iniciales

---

### 💻 Programación
Aplicación en **C# con Windows Forms** basada en POO:

- Clases: Evento, Usuario, Organizador, Ubicacion, Categoria, Inscripcion
- Encapsulación con atributos privados y propiedades
- Lógica de gestión y validaciones

---

## 📄 Documentación
Incluye:
- Instalación de máquina virtual Windows con servidor web
- Diagramas de la base de datos
- Documentación técnica en PDF del sistema

---

## 🔧 Tecnologías utilizadas

- HTML5, CSS3
- MySQL
- C# (.NET Framework)

---

💡 *EcoEventos es un proyecto académico que integra desarrollo web, bases de datos y programación orientada a objetos.*
